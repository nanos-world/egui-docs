### SetText

Sets the text of the button

> Returns the [`Button`](button) (self)

```lua
button:SetText("Text")
```

| Type | Parameter | Default Value | Description |
| :--- | :--- | :--- | :--- |
| `string` | **`text`** |  | The new text |

### GetText

Gets the text of the button

> Returns a `string`

```lua
local text = button:GetText()
```

### SetStyle

Sets the style of the button

> Returns the [`Button`](button) (self)

```lua
button:SetStyle(EGUI.Button.Styles.Rounded)
```

| Type | Parameter | Default Value | Description |
| :--- | :--- | :--- | :--- |
| [`Button Style`](../enums#button-style) | **`style`** |  | The new button style |

### GetStyle

Gets the style of the button

> Returns a string [**`Button Style`**](../enums#button-style)

```lua
local style = button:GetStyle()
```
