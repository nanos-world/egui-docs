### SetValue

Sets the value of the input

> Returns the [`Input`](input) (self)

```lua
input:SetValue("Value")
```

| Type | Parameter | Default Value | Description |
| :--- | :--- | :--- | :--- |
| `string` | **`value`** |  | The new value |

### GetValue

Gets the value of the input

> Returns a `string` of the current value

```lua
local value = input:GetValue()
```

### SetType

Sets the type of the input

> Returns the [`Input`](input) (self)

```lua
input:SetType(EGUI.Input.Types.Password)
```

| Type | Parameter | Default Value | Description |
| :--- | :--- | :--- | :--- |
| [`Input Type`](../enums#input-type) | **`type`** |  | The new input type |

### GetType

Gets the type of the input

> Returns a string [`Input Type`](../enums#input-type)

```lua
local type = input:GetType()
```
