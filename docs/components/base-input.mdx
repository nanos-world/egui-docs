### SetEnabled

Allows enabling/disabling of the input

> Returns the [`Base Input`](base-input) (self)

```lua
input:SetEnabled(false)
```

| Type | Parameter | Default Value | Description |
| :--- | :--- | :--- | :--- |
| `boolean` | **`state`** |  | Whether its enabled |

### IsEnabled

> Returns `boolean` whether its enabled

```lua
if input:IsEnabled() then
    Package.Log("Input is enabled")
else
    Package.Log("Input is disabled")
end
```
